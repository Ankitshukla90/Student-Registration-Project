# Student Registration System

A responsive student registration system built with HTML, CSS, and JavaScript.

# Features
- **Add Student:** Register students with Name, ID, Email, and Contact.
- **Validation:** Ensures valid inputs (e.g., 10-digit contact, valid email).
- **Persistence:** Data is saved in LocalStorage (won't disappear on refresh).
- **Responsive:** Optimized for Mobile, Tablet, and Desktop.
- **Dynamic UI:** Scrollbars appear automatically when data grows.

# Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6)

# How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
